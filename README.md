# **App do jogo JoKenPô** 

> Aplicativo de Android que funcione de forma fácil para o usuário.


## 📱 Descrição

Uma aplicação que é do game JoKenPô consiste o aplicativo em o usuário escolher pedra, pepel ou tesoura e o app vai ver escolher outro, e vai decidir quem ganhou ou empatou.


## 🔧 Funcionalidades da Calculadora de Potencia: 

- [x] Seleção das opções de pedra, papel ou tesoura;
- [x] Sistema de empate ou vitória.     


## 🚀 Tecnologias Utilizadas

- [x] **Android Studio** (Koala | 2024.1.2)
- [x] **Java** para desenvolvimento
- [x] **ConstraintLayout** para interface responsiva
- [x] **TextView** e **EditText** para entrada e exibição de dados
- [x] **Button**   para executar o app.
